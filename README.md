# Rock Paper Scissors Web Game 🎮

An interactive Rock Paper Scissors game built using Python Flask, HTML, CSS, and JavaScript.

## Features
- User vs Computer gameplay
- Random computer choice
- Scoreboard system
- Reset game functionality
- Responsive UI
- Color-based result display

## Tech Stack
- Python (Flask)
- HTML
- CSS
- JavaScript

## How to Run
1. Clone repository
2. Install dependencies
3. Run app

```bash
pip install -r requirements.txt
python app.py
